Checkbox represents an independent choice, commonly one of several values
submitted together. The packaged Primitive owns checked state, activation,
focus, and Indicator visibility. The Recipe adds a one-cell mark, label, and
tone.

Use [Switch](/catalog/switch) for a setting that takes effect immediately and
[Radio Group](/catalog/radio-group) for mutually exclusive options.

## Use the Recipe

### React and Solid

```tsx
import { Checkbox } from "./components/ui/checkbox";

<Checkbox
  label="Enable notifications"
  defaultChecked
  onCheckedChange={(checked) => console.log({ checked })}
/>;
```

Use `checked` with `onCheckedChange` for controlled ownership, or
`defaultChecked` for uncontrolled initial state.

### Core

```ts
import { createCheckbox } from "./components/ui/checkbox";

const checkbox = createCheckbox(ctx, {
  label: "Enable notifications",
  defaultChecked: true,
  onCheckedChange: (checked) => console.log({ checked }),
});

parent.add(checkbox);
```

The factory returns `CheckboxRootRenderable`. The installed source constructs
an Indicator attached to the Root's Store.

## Customize the Recipe

The copied source owns the one-cell mark, label assembly, tone mapping, gap,
and checked-state presentation. Edit it when your design system needs a wider
mark, a different layout, or another convenience prop. Shared glyph and color
defaults come from the [Theme Recipe](/theming).

## Behavior

Checkbox supports controlled or uncontrolled checked state. Enter, Return,
Space, primary-pointer release, and imperative `press()` request the opposite
value. Disabled Checkboxes reject focus and activation.

      type: "string",

      default: '"accent"',
      description: "Selects the primary or success mark color in React and Solid.",
    },
  }}

React and Solid add these fields while forwarding the remaining Root props.
Core adds `label` and `mark`, but not the framework Recipe's `tone` field.

When the mark-and-label structure does not fit, compose the packaged
[Checkbox Primitive](/primitives/checkbox).
