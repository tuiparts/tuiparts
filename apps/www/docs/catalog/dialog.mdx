A modal overlay that contains focus until dismissed. Open state, layer
coordination, dismissal, and focus restoration come from the packaged
primitive. The backdrop, popup layout, colors, and controls install as source
you edit.

## Usage guidelines

- **Use `Dialog` for a short task that requires the user's attention.** Prefer
  inline controls when the user can complete the task without leaving the
  current context.
- **Give every dialog a visible title.** Add a description when the action or
  its consequences need more context.
- **tuiparts contains focus while the dialog is open.** You are responsible
  for placing the actions in a clear order and rendering visible focus states.

## Installation

<CodeGroup>

```bash React
pnpm dlx shadcn@4.13.0 add @tuiparts/react/dialog
```

```bash Solid
pnpm dlx shadcn@4.13.0 add @tuiparts/solid/dialog
```

```bash Core
pnpm dlx shadcn@4.13.0 add @tuiparts/core/dialog
```

</CodeGroup>

The command above copies the recipe into `components/ui`. The file belongs to
your application.

## Anatomy

Import the installed components and assemble the dialog:

```tsx title="app/settings-dialog.tsx"
import {
  Dialog,
  DialogClose,
  DialogContent,
  DialogDescription,
  DialogTitle,
  DialogTrigger,
} from "./components/ui/dialog";

<Dialog>
  <DialogTrigger>
    <text content="Open settings" />
  </DialogTrigger>
  <DialogContent>
    <DialogTitle content="Settings" />
    <DialogDescription content="Change the options for this project." />
    <DialogClose>
      <text content="Close" />
    </DialogClose>
  </DialogContent>
</Dialog>;
```

The recipe composes `Portal`, `Backdrop`, and `Popup` into `DialogContent`.
Compose the packaged parts directly when you need a different layer structure.

## Keyboard interactions

| Key | Behavior |
| --- | --- |
| `Esc` | Dismisses the topmost open dialog. |
| `Tab` | Moves focus to the next focusable control inside the dialog. |
| `Shift+Tab` | Moves focus to the previous focusable control inside the dialog. |
| `Enter` | Activates the focused `Trigger` or `Close`. |
| `Space` | Activates the focused `Trigger` or `Close`. |

Focus wraps within the topmost dialog. Closing it restores focus to its
trigger. The primary pointer button activates `DialogTrigger` and
`DialogClose`; pressing the backdrop dismisses the topmost dialog.
