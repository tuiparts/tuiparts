Dialog presents a short modal task or decision. The packaged Primitive owns
open state, renderer-scoped layers, topmost dismissal, focus containment, and
restoration. The Recipe provides a centered popup, backdrop, border, spacing,
and styled controls.

Prefer inline controls when the task does not need to interrupt the current
context. Give every Dialog a visible title and add a description when the
action or consequences need explanation.

## Use the Recipe

### React and Solid

```tsx
import {
  Dialog,
  DialogClose,
  DialogContent,
  DialogDescription,
  DialogTitle,
  DialogTrigger,
} from "./components/ui/dialog";

<Dialog>
  <DialogTrigger>
    <text content="Open settings" />
  </DialogTrigger>
  <DialogContent>
    <DialogTitle content="Settings" />
    <DialogDescription content="Change this project's options." />
    <DialogClose>
      <text content="Close" />
    </DialogClose>
  </DialogContent>
</Dialog>;
```

`DialogContent` composes the Primitive's Portal, Backdrop, and Popup Parts.

### Core

The Core Recipe returns the Parts needed for imperative assembly:

```ts
import { TextRenderable } from "@opentui/core";
import {
  addDialogClose,
  addDialogDescription,
  addDialogTitle,
  createDialog,
} from "./components/ui/dialog";

const dialog = createDialog(ctx);
dialog.trigger.add(new TextRenderable(ctx, { content: "Open settings" }));
addDialogTitle(ctx, dialog, "Settings");
addDialogDescription(ctx, dialog, "Change this project's options.");
addDialogClose(ctx, dialog, "Close");

parent.add(dialog.root);
parent.add(dialog.portal);
```

`createDialog()` returns `root`, `trigger`, `portal`, `backdrop`, and `popup`.
Core callers add both the Root and Portal to the render tree. The helper
functions append semantic text and Close Parts to the Popup.

## Customize the Recipe

The copied source owns the relationship between `DialogContent`, Portal,
Backdrop, and Popup along with popup width, border, spacing, and control
presentation. Edit that source when you need another layer structure or
convenience API. Shared colors and borders come from the
[Theme Recipe](/theming).

## Behavior

Dialog supports controlled or uncontrolled open state. It coordinates
topmost Escape and Backdrop dismissal, contains Tab focus within the active
Popup, and restores focus when the layer closes. Dismissal details report the
reason and allow cancelable transitions to be prevented.

## Recipe API

React and Solid `DialogContent` accepts Primitive Popup props and adds
`backdropColor?: ColorInput`. Other Recipe exports add no fields; they forward
the props of their matching Primitive Parts.

Core adds `width` to `DialogOptions`. Its helper functions assemble and append
the Recipe's title, description, and close presentation.

For a different Portal, Backdrop, or Popup structure, compose the packaged
[Dialog Primitive](/primitives/dialog).
