Every recipe in the starter catalog is available as `core/<name>`,
`react/<name>`, and `solid/<name>`, installs through the shadcn CLI, and
becomes source your application owns.

**Recipes are starting points.** The palette, density, symbols, labels, and
intent variants in each installed file are examples for you to edit, not
package APIs.
