title: Switch
description: A checked setting rendered as a movable thumb over a glyph track.

Switch represents a setting that takes effect immediately. The packaged
Primitive owns checked state, activation, focus, and the always-mounted Thumb
Part. The Recipe chooses the track, thumb glyph, density, label, and colors.

Use [Checkbox](/catalog/checkbox) for choices commonly submitted together.

## Use the Recipe

### React and Solid

```tsx
import { Switch } from "./components/ui/switch";

<Switch
  label="Dark mode"
  density="comfortable"
  symbols="round"
  defaultChecked
/>;
```

Use `checked` with `onCheckedChange` for controlled ownership, or
`defaultChecked` for uncontrolled initial state.

### Core

```ts
import { createSwitch } from "./components/ui/switch";

const colorMode = createSwitch(ctx, {
  label: "Dark mode",
  density: "comfortable",
  defaultChecked: true,
});

parent.add(colorMode);
```

The factory returns `SwitchRootRenderable` and constructs a Thumb attached to
the Root's Store.

## Customize the Recipe

The copied source owns the track and thumb glyphs, track width, Thumb
positioning, label gap, density choices, and colors. Edit those choices or add
variants directly in the installed file. Shared defaults come from the
[Theme Recipe](/theming).

## Behavior

Switch supports controlled or uncontrolled checked state. Enter, Return,
Space, primary-pointer release, and imperative `press()` request the opposite
value. Disabled Switches reject focus and activation.

## Recipe API

<TypeTable
  type={{
    label: {
      type: "string",
      required: true,
      description: "Text rendered next to the track.",
    },
    density: {
      type: '"compact" | "comfortable"',
      default: '"compact"',
      description: "Uses a three-cell or five-cell track and adjusts the label gap.",
    },
    symbols: {
      type: '"round" | "ascii"',
      description: "Overrides the theme thumb and track glyphs with a named pair.",
    },
  }}
/>

React and Solid add these fields while forwarding the remaining Root props.
Core adds the same three Recipe fields through `createSwitch()`.

To design a different track, Thumb position, or public API, compose the
packaged [Switch Primitive](/primitives/switch).
