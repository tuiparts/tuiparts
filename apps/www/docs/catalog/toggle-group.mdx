Toggle Group coordinates related two-state actions. The packaged Primitives
own single or multiple selection, item registration, orientation-aware roving
focus, and disabled behavior. The Recipe adds group layout and labeled Toggle
presentation.

Use standalone [Toggle](/catalog/toggle) for one independent stateful action
and [Radio Group](/catalog/radio-group) when only one value may be selected.

## React and Solid

```tsx
import {
  ToggleGroup,
  ToggleGroupItem,
} from "./components/ui/toggle-group";

<ToggleGroup defaultValue={["left"]}>
  <ToggleGroupItem value="left" label="Left" />
  <ToggleGroupItem value="center" label="Center" />
  <ToggleGroupItem value="right" label="Right" />
</ToggleGroup>;
```

The Recipe uses a horizontal layout unless `orientation="vertical"` is set.
Set `multiple` to allow more than one pressed value.

## Core

```ts
import {
  createToggleGroup,
  createToggleGroupItem,
} from "./components/ui/toggle-group";

const alignment = createToggleGroup(ctx, {
  defaultValue: ["left"],
});

alignment.add(
  createToggleGroupItem(ctx, alignment.store, {
    value: "left",
    label: "Left",
  }),
);
alignment.add(
  createToggleGroupItem(ctx, alignment.store, {
    value: "center",
    label: "Center",
  }),
);

parent.add(alignment);
```

Each Core item receives the group's Store explicitly.

## Selection

`value`, `defaultValue`, and `onValueChange` always use arrays. In the default
single-selection mode, only the first unique value is retained and pressing
the active item can clear the group. With `multiple`, every unique active
value is retained.

Focus movement does not change selection.

## Keyboard interaction

| Key | Behavior |
| --- | --- |
| `Left` / `Right` | Moves focus in a horizontal group. |
| `Up` / `Down` | Moves focus in a vertical group. |
| `Home` | Moves focus to the first available Toggle. |
| `End` | Moves focus to the last available Toggle. |
| `Enter`, `Return`, or `Space` | Changes the focused Toggle's pressed state. |

Unavailable items are skipped. Navigation wraps by default; set
`loopFocus={false}` to stop at the collection edges.

### ToggleGroupItem

React and Solid items require `value` and `label`, and otherwise extend Toggle
Primitive props except for `children` and `value`. The Core
`ToggleGroupItemOptions` starter surface contains `value`, `label`, and
`disabled`; edit the installed factory to expose additional item callbacks.
