Toggle Group coordinates related two-state actions. The packaged Primitives
own single or multiple selection, item registration, orientation-aware roving
focus, and disabled behavior. The Recipe adds group layout and labeled Toggle
presentation.

Use standalone [Toggle](/catalog/toggle) for one independent stateful action
and [Radio Group](/catalog/radio-group) when only one value may be selected.

## Use the Recipe

### React and Solid

```tsx
import {
  ToggleGroup,
  ToggleGroupItem,
} from "./components/ui/toggle-group";

<ToggleGroup defaultValue={["left"]}>
  <ToggleGroupItem value="left" label="Left" />
  <ToggleGroupItem value="center" label="Center" />
  <ToggleGroupItem value="right" label="Right" />
</ToggleGroup>;
```

The Recipe uses a horizontal layout unless `orientation="vertical"` is set.
Set `multiple` to allow more than one pressed value.

### Core

```ts
import {
  createToggleGroup,
  createToggleGroupItem,
} from "./components/ui/toggle-group";

const alignment = createToggleGroup(ctx, {
  defaultValue: ["left"],
});

alignment.add(
  createToggleGroupItem(ctx, alignment.store, {
    value: "left",
    label: "Left",
  }),
);
alignment.add(
  createToggleGroupItem(ctx, alignment.store, {
    value: "center",
    label: "Center",
  }),
);

parent.add(alignment);
```

Each Core item receives the group's Store explicitly.

## Customize the Recipe

The copied source owns group layout and its labeled `ToggleGroupItem`
presentation. Edit it to change item content, selected styling, spacing, or
the convenience props forwarded by the Core factories. Shared colors and
density come from the [Theme Recipe](/theming).

## Behavior

Toggle Group supports controlled or uncontrolled array-valued selection in
single or multiple mode. Arrow keys, Home, and End move the roving focus
without changing selection; activating the focused Toggle changes its pressed
state. Unavailable items are skipped, and navigation can wrap at collection
edges.

## Recipe API

`ToggleGroup` introduces no convenience fields. It forwards Primitive group
props while using `orientation` to choose a row or column starter layout.

`ToggleGroupItem` adds required `label` presentation and requires `value` for
group identity. React and Solid forward the remaining Toggle props. Core also
accepts `disabled`; edit the installed factory to expose additional item
callbacks.

For a different item structure or design-system API, compose
[Toggle Group and Toggle](/primitives/toggle-group) directly.
