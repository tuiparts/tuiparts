# Switch Recipe

The Core, React, and Solid recipes are editable consumer-owned presentation
for the packaged `Switch` Root/Thumb behavior. They choose track and thumb
glyphs, compact and comfortable densities, positioning, colors, and labels.

Install the framework-specific `core/switch`, `react/switch`, or
`solid/switch` registry item. Primitive packages retain checked ownership,
focus, activation, disabled behavior, and the always-mounted Thumb lifecycle.
